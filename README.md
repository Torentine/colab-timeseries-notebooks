# Colab/GitHub ready notebooks

Папка готова для загрузки на GitHub и запуска в Google Colab.

## Состав

- `01_etth1_time_series_colab.ipynb` - анализ и прогноз ETTh1.
- `02_electricity_forecasting_colab.ipynb` - анализ и прогноз потребления электроэнергии.
- `data/etth1_ot_prepared.csv` - данные для первого ноутбука.
- `data/electricity_daily_prepared.csv` - данные для второго ноутбука.

## Как запускать в Colab

1. Загрузите эту папку на GitHub.
2. Откройте нужный `.ipynb` через Google Colab.
3. Запустите ячейки сверху вниз.

Ноутбуки показывают результаты внутри ячеек и не сохраняют отчеты, CSV или картинки в отдельные папки.
